## How to run

### 1. Run `midas-http`

```bash
cargo run --bin midas-http
```

### 2. Run `midas-web`

```bash
cd midas-web
npm install
npm run dev
```

### ?. Optional

You can update Index Data using `midas-spider`

```bash
cargo run --bin midas-spider
```

## Test

```bash
cargo test
```
